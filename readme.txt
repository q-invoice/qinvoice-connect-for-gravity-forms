Connects your Gravity Forms forms to q-invoice for automatic invoicing.

== Description ==

Easily generate and send professional looking invoices or estimates for every Gravity Forms entry through q-invoice. Invoices can be saved as draft for later editing or sent directly to your customers' emailaddress. Different layouts can be used for invoices and estimates.

A subscription for q-invoice.com is required. 

== Installation ==

Install the plugin by uploading the zipfile in your WP admin interface or via FTP:

1. Upload the folder `gravityforms-qinvoice-connect` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Find the configuration page and fill out your settings and preferences
